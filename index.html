achievements
PlatformIO
Professional collaborative platform for embedded development

PlatformIO is a cross-platform, cross-architecture, multiple framework, professional tool for embedded systems engineers and for software developers who write applications for embedded products.

Kicad
A Cross Platform and Open Source Electronics Design Automation Suite

KiCad is an open source software suite for Electronic Design Automation (EDA). The programs handle Schematic Capture, and PCB Layout with Gerber output. The suite runs on Windows, Linux and macOS and is licensed under GNU GPL v3.

Arduino
open-source electronics platform based on easy-to-use hardware and software

A worldwide community of makers has gathered around this open-source platform, their contributions have added up to an incredible amount of accessible knowledge that can be of great help to novices and experts alike.

MicroPython
a lean and efficient implementation of the Python 3

MicroPython is packed full of advanced features such as an interactive prompt, arbitrary precision integers, closures, list comprehension, generators, exception handling and more. Yet it is compact enough to fit and run within just 256k of code space and 16k of RAM.

ThingsBoard
